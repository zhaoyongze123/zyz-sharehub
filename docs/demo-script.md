# ShareHub 演示脚本

## 演示流程

### 1. 首页开场

- 打开首页
- 说明产品定位、核心用户和主导航入口
- 展示“开始使用”和“看看路线图”

### 2. 资源模块

- 进入资料广场
- 打开一条真实资料详情
- 说明真实数据读取、相关推荐和详情页信息结构

### 3. 路线模块

- 进入学习路线图
- 打开一条真实路线
- 点击节点，展示节点详情和关联资源跳转

### 4. 笔记模块

- 打开社区笔记页
- 进入一条真实笔记详情
- 展示正文、目录和状态信息

### 5. 个人中心

- 从登录回调进入 `/me`
- 展示真实聚合数据、头像上传、只读资料提示
- 打开简历工作台，说明当前简历按纯前端模块维护

### 6. 后台治理

- 切换管理员身份
- 打开管理中心仪表盘
- 依次展示举报、审计日志、审核、用户、标签分类
- 再切回普通用户，演示权限拦截

### 7. 发布链路

- 进入资料发布页，上传附件并提交审核
- 从结果卡进入真实资料详情页
- 进入路线创建页，先提交主体，再追加真实节点
- 从结果卡进入真实路线详情页

## 演示讲解词重点

- 前台核心读写链路已经走真实接口
- 后台治理链路已经具备真实读取和核心操作
- 夜间自动化可持续推进、回归和通知
- 简历模块当前按纯前端产品边界维护

## 演示前准备

- 后端已启动并通过健康检查
- 前端已启动
- `tests/e2e/full-site-walkthrough.spec.ts` 最近一次于 2026-04-10 07:00 +0800 在 `http://127.0.0.1:14173` + `http://127.0.0.1:18080` 环境下 7/7 通过（13.1s，本轮夜间复核）
- 若要现场演示后台治理，准备管理员登录态

## 异常预案

- 如果接口超时：先展示健康检查与日志
- 如果页面空态：切换到预置数据或重新触发 smoke 种子数据
- 如果后台权限异常：重新注入管理员本地登录态
