# Pixso 对照验收清单

## 结构一致性
- [ ] 页面模块顺序与 page-contract 对齐
- [ ] 关键组件使用 component-contract 定义状态

## 视觉一致性
- [ ] 颜色全部引用 design-token（无硬编码）
- [ ] 间距、圆角、阴影均引用 design-token
- [ ] 关键区域（头部/主内容/表单/卡片/弹窗）高保真对齐

## 交互一致性
- [ ] 默认/hover/disabled/loading 状态完整
- [ ] 空态、错误态、权限受限态与设计稿一致

## 响应式一致性
- [ ] 桌面与移动断点按 token 执行
- [ ] 主流程页面在主流浏览器显示一致
