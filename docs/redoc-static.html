ShareHub API (0.2.0)

Download OpenAPI specification:

ShareHub 后端标准接口描述。 当前大部分模块仍基于内存存储和模拟行为实现，联调前请同时阅读 docs/backend-api-reference.md。

Responses

本地开发环境

  • "data": {
    • "loginUrl": "/oauth2/authorization/github"
    },
  • "message": "string"
}
